About Wilfred Mukiri

Wilfred Mukiri is a seasoned software developer with over 10 years of experience in the industry. He has worked on a wide range of projects and has experience in various programming languages and technologies. He is passionate about teaching and sharing his knowledge with others. He is currently an instructor at the School of Advanced Programming (SOAP) where he teaches courses on web development, software engineering, and other ICT fields.

Courses Taught by Wilfred Mukiri

Web Development

This course covers the basics of web development including HTML, CSS, and JavaScript. It also covers advanced topics such as responsive design, web performance optimization, and web security.

Software Engineering

This course covers the principles of software engineering including software design, software testing, and software maintenance. It also covers advanced topics such as software architecture, software development methodologies, and software project management.

Software Applications

This course covers the basics of software applications including Microsoft Office, Google Workspace, and other popular software applications. It also covers advanced topics such as software customization, software integration, and software automation.

Graphics Design

This course covers the basics of graphics design including Adobe Photoshop, Adobe Illustrator, and other popular graphics design tools. It also covers advanced topics such as graphic design principles, graphic design techniques, and graphic design trends.
